# Vrinda_Store_Data_Analysis_Excel_Project
Excel dashboard and sales data analysis project

# Vrinda Store Annual Report 2022

## Project Overview
This project contains an Excel dashboard for analyzing Vrinda Store sales data.

## Features
- Order vs Sales Analysis
- Sales: Men vs Women
- Order Status Analysis
- Top 5 States Sales
- Orders: Age vs Gender
- Sales Channel Analysis

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers
- Dashboard Design
